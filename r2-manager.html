ℹ️ Cara Mendapatkan Kredensial
1. Buka dash.cloudflare.com
2. Pilih akun → R2 Object Storage
3. Salin Account ID dari halaman Overview
4. Klik Manage R2 API Tokens → Buat token baru
5. Permission: Object Read & Write
6. Salin Access Key ID dan Secret Access Key
